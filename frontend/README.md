# AutoMind AI Frontend

## Purpose

The frontend will provide the user interface for AutoMind AI. It is planned to use React, Vite, and TypeScript for future vehicle status views, assistant interactions, navigation, shared layouts, and client-side services.

This frontend currently contains structure and package metadata only.

## Folder Structure

- `src/assets/` - Future images, icons, fonts, and other static assets
- `src/components/` - Future reusable UI components
- `src/hooks/` - Future reusable React hooks
- `src/layouts/` - Future page and application layouts
- `src/pages/` - Future route-level pages
- `src/routes/` - Future route definitions
- `src/services/` - Future frontend service modules
- `src/types/` - Future TypeScript type definitions
- `src/utils/` - Future frontend utility functions

## Start Later

Frontend application files have not been created yet. Once the Vite application is implemented, install dependencies and start development with:

```bash
npm install
npm run dev
```
